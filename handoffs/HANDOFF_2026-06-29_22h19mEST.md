## Decisions locked
- **Static HTML is the accessible deliverable; DPG (`app_base.py`) is prototype-only** — DPG's lack
  of an accessibility tree is architectural, not themeable. Don't try to make DPG WCAG AA.

  contrast inside one tab; rejected the commented-out-with-asserted-error alternative (errors must
  be real g++ output).

  border (never color alone).
- **OpenSpec governance:** record retroactive work in plain English ("record this in OpenSpec");
  use `/opsx:propose` only for spec-first *new* features. Commit in scoped logical groups on `main`.
